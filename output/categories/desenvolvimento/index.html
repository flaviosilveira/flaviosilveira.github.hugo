Entenda como funciona o PHP! (Do código fonte até a renderização)

Esse artigo é quase uma tradução fiel do artigo How PHP Executes from Source Code to Render de Thomas Punt, revisado por Younes Rafie. No original o autor coloca os pedaços de códigos para você testar por conta. Aqui, além disso, adicionei como rodar os mesmos com um único comando, usando docker. Assim você não precisa de muito esforço para ver os exemplos. Criei uma imagem no dockerhub com tudo necessário para rodar os exemplos desse artigo.
